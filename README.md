# E-commerce Sales Analytics Project

## What it does
- Uses SQL to analyze e-commerce sales data  
- Shows revenue, top products, RFM, cohort analysis, and customer lifetime value  

## How to run
1. Create database: `CREATE DATABASE ecommerce_project;`  
2. Run `schema.sql` → creates tables  
3. Run `insert_data.sql` → adds sample data  
4. Run `cleaning.sql` → cleans data  
5. Run queries in `sql/analysis/` → see outputs in `results/`  

## Tools
- MySQL Workbench  
- SQL
